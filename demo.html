Validation

This is just another jQuery validation plugin, positive points are:

  • Easy to use.
  • Easy to extend.
  • Extend everything, add your own validation and output functions.
  • Generated html is valid, the code itself validates in jslint (good parts on except use strict).
  • Uses title attribute for textual errors, can use classes for custom behavior.
  • Works with jQuery 1.3.2 and 1.4
  • Can also be used on single elements outside of forms and inside other (ajax) functions.

Download here

Usage

Basic usage is very simpel, just create a form, give the required elements the validate_required class and put $("form").validate() in the bottom of your page.

Methods

validate       Validate the elements that are required. (Can be called by hand for ajax/other functions and single elements)

Options

(very limited overview atm, more will come later, I promise)

required_class   The class the validation plugin identifies the required elements with
Default: validate_required
error_attribute   The attribute that holds the error message
Default: title
error_output   The default output types for each type of field
Default: {text:"class", email:"class", select:"class", textarea:"class", other:"append"}
error_class   The class invalid elements get when they are invalid
Default: validate_invalid
error_append_char   The character that gets appended after invalid elements
Default: *
error_append_class   The class of the character that gets appended after invalid elements
Default: error_append_class
error_parent_class   The class of the parent of the invalid elements
Default: error_parent_class
types   List of types of elements to validate and the function to validate them
Default: {text:validate_text, email:validate_email, checkbox:validate_checkbox, radio:validate_radio, select:validate_select, textarea:validate_text}
output   List of output modules and the function to call them
Default: {class:output_class, inline:output_inline, append:output_append, alert:output_alert, parent:output_parent}

Add custom validation function

Code (click here to show/hide) 
#
# html:
#
<form id="form4" method="post" action="./demo.html">
<table>
	<tr>
		<th>Please enter one's and zero's only</th>
		<td><input name="foo" type="text" class="validate_required validate_type_binary"/></td>
	</tr>
	<tr>
		<td></td>
		<td><input type="submit" value="Submit"/></td>
	</tr>
</table>
</form>

#
# javascript:
#
$("#form4").validate({
	// Define the type of validation to use
	error_output:	{
		binary:			"append"
	},
	// Define the validation function
	types:			{
		binary:		function (element, object) {
			var valid	=	true,
			pattern		=	/[^0|1]/;
			
			if (pattern.exec(element.el.val()) || element.el.val().length < 1) {
				valid		=	false;
			}
			
			return valid; 
		}
	}
});
Please enter one's and zero's only

Add custom output function

Code (click here to show/hide) 
#
# html:

<form id="form5" method="post" action="./demo.html">
<table>
	<tr>
		<th>Enter text please</th>

	</tr>

		<td></td>
		<td><input type="submit" value="Submit"/></td>
	</tr>

#
# javascript:
#
$("#form5").validate({
	// Define the type of error output to use
	error_output:	{
		text:			"fadeInOut"
	},
	// Define the output function
	output:			{
		fadeInOut:	function (element, object) {
			
			var i,
			fadeIn	=	function(i){
				element.el.fadeIn(100, function(){
					i++;
					if(i < 20){
						fadeOut(i);
					}
				});
			},
			fadeOut	=	function(i){
				element.el.fadeOut(100, function(){
					i++;
					if(i < 20){
						fadeIn(i);
					}
				});
			};
			fadeOut(0);
			
		}
	}
});
